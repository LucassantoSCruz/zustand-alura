# Fokus - App de Cronômetro (Zustand)

Este é um pequeno projeto desenvolvido com base no curso da Alura, onde aprendi a usar o Zustand para gerenciamento de estado global em aplicações React.

## O que o app faz

Possui 3 modos de cronômetro: Foco, Descanso Curto e Descanso Longo

Permite iniciar e pausar o timer

Altera cores e textos da interface conforme o modo selecionado

Usa Zustand para controlar o estado de forma simples e eficiente

## Tecnologias

React

Zustand

JavaScript

CSS
